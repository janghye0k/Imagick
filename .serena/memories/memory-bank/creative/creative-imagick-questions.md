## C. 도구별 화면·플로우

| 번호 | 질문              | 답변                                                                                                                                                                                                                                       |
| ---- | ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| C1   | **파일 선택**     | **(3)** 드래그 앤 드롭 + 클릭. **전역 동작**: 모든 페이지에서 드래그 앤 드롭 시 항상 파일 업로드. 랜딩·편집과 무관한 페이지에서 업로드 시 **Convert**로 자동 이동하여 진행. 페이지 전환 시에도 **업로드된 파일 유지**. UI/UX는 자연스럽게. |
| C2   | **미리보기 위치** | 구현 시 **데스크톱·모바일 각각 자연스러운** 방식으로 판단 적용 (데스크톱: 옵션 오른쪽 또는 아래, 모바일: 스택/탭 등).                                                                                                                      |
| C3   | **다운로드 버튼** | C2 레이아웃에 맞춰 적용.                                                                                                                                                                                                                   |
| C4   | **여러 파일**     | **MVP부터 다중 파일 지원**. 단, **최대 개수**와 **최대 용량** 제한 도달 시 추가 업로드 불가. (구체값: creative-imagick-uiux.md 참고)                                                                                                       |
| C5   | **Resize 옵션**   | **가능한 모든 Resize** — 너비/높이(px), 비율(%), 가로/세로 비율 유지, 프리셋 등.                                                                                                                                                           |
| C6   | **Convert 포맷**  | **PNG, JPG, GIF, WebP, AVIF** 등 브라우저에서 가능한 포맷 전부.                                                                                                                                                                            |
| C7   | **Compress 품질** | **둘 다** — 슬라이더(0–100) + 단계(낮음·중간·높음).                                                                                                                                                                                        |
| C8   | **Filter 종류**   | **가능한 모두** — 밝기, 대비, 채도, 그레이스케일, 블러, 세피아, 색온도, 반전 등.                                                                                                                                                           |
